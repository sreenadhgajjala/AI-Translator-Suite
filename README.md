# 🌍 Sreenadh AI Translator Suite

## 🚀 Overview

Sreenadh AI Translator Suite is a modern AI-powered language translation and writing assistant developed using Python and Streamlit. The application supports 100+ languages and provides advanced NLP features such as translation, language detection, text-to-speech, grammar correction, paraphrasing, translation history, and analytics.

---

## ✨ Features

### 🌐 Language Translation

* Translate text into 100+ languages
* Automatic language detection
* Fast and accurate translations

### 🔊 Text-to-Speech

* Convert translated text into speech
* Audio playback support
* Download MP3 audio files

### ✍️ Grammar Checker

* Detect grammar mistakes
* Correct spelling and sentence structure
* Improve writing quality

### 🔁 AI Paraphraser

* Rewrite text while preserving meaning
* Generate alternative sentence structures
* Improve readability

### 📜 Translation History

* Save translation records
* Download translation history
* Clear history functionality

### 📊 Analytics Dashboard

* Word count
* Character count
* Translation usage insights

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/sreenadhgajjala/AI-Translator-Suite.git
```

### Navigate to Project

```bash
cd AI-Translator-Suite
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

Windows:

```bash
venv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
streamlit run app.py
```
